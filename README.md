# EEOB 5460 Python Assignment

This repository contains my completed Python assignment for EEOB 5460.

## Overview

This project analyzes cytochrome-b sequences from penguin species. The workflow involves:

- Translating DNA sequences into amino acid sequences
- Calculating GC content and molecular weight
- Combining sequence data with body mass data
- Making visualizations to see relationships

## Files

- `Sirotiak_EEOB5460_Spring2026.ipynb` – main analysis Jupyter notebook 
- `penguins_mass.csv` – body mass data 
- `penguins_cytb.fasta` – DNA sequences 
- `sequence_translate.py` – original script
- `penguins_mass_cytb.csv` – final output data

## How to Run

Open the Jupyter notebook and run all cells from top to bottom.
